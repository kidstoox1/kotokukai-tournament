# 日本拳法大会運営システム — 詳細仕様書

## 1. 画面一覧と役割

| 画面 | URL | 利用者 | 端末 | 接続方式 |
|------|-----|--------|------|----------|
| 管理ダッシュボード | `/` | 大会運営者 | PC | Realtime |
| 記録係入力 | `/referee` | 各コートの記録係 | タブレット | Realtime |

| 観覧用 | `/spectator` | 保護者・観客 | スマホ | ポーリング（30秒） |

## 2. 大会運営フロー

```
[事前準備]
1. 管理者: 応募フォームから選手データ取り込み
2. 管理者: カテゴリ自動振り分け（学年×性別）
3. 管理者: 各カテゴリの開始形式を選択
   - リーグ戦（デフォルト）
   - トーナメント
   - 予選トーナメント
4. 管理者: 各カテゴリのコート割り当て（A/B/C/D）
5. 管理者: 進出人数（デフォルト1名）、3位決定戦（デフォルトON）を設定
6. 管理者:「全カテゴリ一括開始」or 個別に「開始」

[リーグ戦フェーズ]
7. システム: グループ自動分け（3〜4人/グループ）
8. 記録係: 試合ごとに「開始」→ 結果入力 →「確定」
9. 管理者: 全試合完了後「次ステージへ」を選択
   - リーグ決勝（上位者で総当たり → 順位確定）
   - 決勝トーナメント（上位者でトーナメント）
   ※ 誤操作時は「← 戻す」で前フェーズに戻せる

[リーグ決勝フェーズ]
10. 各グループ上位者が1グループで総当たり
11. 全試合完了 → 勝ち点+本数差で順位確定
12. 同率の場合は同順位（全員🥇 等）

[トーナメントフェーズ]
13. シングルエリミネーション（BYE自動配置）
14. 3位決定戦:
    - 4名以上のブラケット → 準決勝敗者同士
    - 2名のブラケット（決勝のみ）→ 各グループ2位同士
15. 試合順: 3位決定戦 → 決勝の順で進行

[完了]
16. 最終結果パネルに🥇🥈🥉を表示
```

## 3. 試合記録仕様

### 3.1 記録項目

| 項目 | 値の範囲 | 説明 |
|------|---------|------|
| 赤の本数 | 0, 1, 2 | 赤が取った本数 |
| 白の本数 | 0, 1, 2 | 白が取った本数 |
| 赤への警告 | 0, 1, 2, 3 | 赤に出された警告回数 |
| 白への警告 | 0, 1, 2, 3 | 白に出された警告回数 |
| 結果タイプ | normal / draw / default_win / disqualification | |

### 3.2 スコア計算ロジック

```
最終スコアA = 入力本数A + floor(白への警告 / 2)
最終スコアB = 入力本数B + floor(赤への警告 / 2)

例: 赤が1本取得、白に警告2回 → 赤の最終スコア = 1 + 1 = 2
```

### 3.3 結果タイプ別処理

| タイプ | スコア | 勝者 |
|--------|--------|------|
| 通常 | 入力値 + 警告ボーナス | スコアが高い方 |
| 引き分け | そのまま | なし（リーグ戦のみ） |
| 不戦勝 | 2-0（勝者側を選択） | 選択された側 |
| 失格 | 0-2（失格側を選択） | 失格でない方 |

### 3.4 結果の修正

- 完了済み試合に「修正」ボタンを表示
- クリックすると前回の入力値がセットされた状態でモーダルが開く
- トーナメント試合を修正した場合、次ラウンドの選手も自動更新

## 4. リーグ戦仕様

### 4.1 グループ分け

```
参加者6人以下 → 3人/グループ
参加者7人以上 → 4人/グループ
同じ道場の選手はできるだけ別グループに（ベストエフォート）
```

### 4.2 順位計算

| 項目 | 値 |
|------|-----|
| 勝利 | 3点 |
| 引き分け | 1点 |
| 敗北 | 0点 |

**ソート順**: 勝ち点 降順 → 本数差（得本-失本）降順

### 4.3 同率順位

勝ち点 AND 本数差が完全に同じ場合 → 同順位
```
例: 3人とも 3pt, 本数差0 → 全員1位
例: 1人 6pt, 2人 3pt 本数差0 → 1位, 2位タイ, 2位タイ
```

## 5. トーナメント仕様

### 5.1 ブラケット生成

- 参加者数を2の累乗に切り上げ（例: 6人→8枠）
- 不足分はBYE（不戦勝）を配置
- BYE勝者は自動的に次ラウンドに進出

### 5.2 3位決定戦の2パターン

**パターンA: 4名以上のブラケット（準決勝あり）**
- 準決勝の敗者同士が3位決定戦
- 準決勝の結果入力時に敗者が自動セット

**パターンB: 2名のブラケット（決勝のみ）**
- リーグ戦の各グループ2位同士が3位決定戦
- リーグ→トーナメント移行時に選手が自動セット

### 5.3 試合順序

3位決定戦 → 決勝の順に表示・進行

## 6. フェーズ遷移

```
[setup] → [league] → [league_final] → (完了)
                   → [final_tournament] → (完了)
         → [final_tournament] → (完了)
         → [pre_tournament] → [final_tournament] → (完了)

戻り操作:
[league_final] → [league] （リーグ決勝データ削除）
[final_tournament] → [league] （トーナメントデータ削除）
[pre_tournament] → [league] （トーナメントデータ削除）
```
