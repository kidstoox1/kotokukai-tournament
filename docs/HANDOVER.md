> 最終更新: 2026-03-08

## 1. プロジェクト概要

日本拳法孝徳会（尼崎市）の大会をリアルタイムで運営管理するWebアプリ。
4コート同時進行のトーナメント・リーグ戦を、記録係がタブレットで入力し、保護者がスマホで閲覧できるシステム。

- **大会規模**: 小学生〜中学生、約150名、16カテゴリ、4コート
- **プロジェクトパス**: `C:/Users/kidst/FREESTYLE CREATION Dropbox/another/クライアント/日本拳法孝徳会/tournament-system/`

## 4. 現在の状態（Phase 1 完了）

### ビルド状況
- **コンパイル**: OK
- **型チェック**: OK（`store/tournament.ts:212`の型エラーは修正済み）
- **`.next`ロックエラー**: Dropbox同期による無害なエラー（コード問題ではない）

### 実装済み機能

#### 管理画面
- サンプルデータ読込（150名16カテゴリ）
- カテゴリごとの設定: 開始形式（リーグ戦/トーナメント/予選トーナメント）、コート割当（A〜D）、進出人数（1〜3名）、3位決定戦の有無
- 全カテゴリ一括開始
- フェーズ進行・巻き戻し

#### 記録係入力画面
- コート選択（A〜D）
- 試合入力: 本数0/1/2、警告0〜3回、引き分け、不戦勝、失格
- 警告2回→相手に1本の自動加算（プレビュー付き）
- 試合結果の修正機能
- 3位決定戦を決勝より先に表示するソート

#### 会場モニター
- 4コート同時表示（2x2グリッド）
- 各コートの進行率と現在の対戦表示

#### 観覧用ビュー（保護者向け）
- カテゴリ選択 → リーグ順位表・トーナメント表の閲覧
- 最終結果表示

#### 試合ロジック
- リーグ戦（総当たり）→ 勝ち点制（勝利3点・引分1点）
- リーグ決勝（上位者で総当たり → 順位確定、同率順位対応）
- トーナメント（シングルエリミネーション + BYE + 3位決定戦）
- 3位決定戦: 準決勝敗者同士 or 各グループ2位同士（ブラケットサイズに応じて自動判定）

## 7. 既知の問題・注意点

| 項目 | 詳細 |
|------|------|
| `page.tsx`の肥大化 | 31000+トークンの巨大ファイル。読み込み時は分割読み込み必要。将来的にコンポーネント分割推奨 |
| Dropboxロック | `.next`ディレクトリがDropbox同期でロックされることがある。ビルド時のEBUSYエラーは無害 |
| DB未接続 | 現在はブラウザ内のZustandストアのみ。リロードでデータ消失 |
| 同時接続制限 | Supabase無料枠200接続。保護者はポーリング方式で対応予定 |
