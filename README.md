# 日本拳法大会運営システム

## 初回セットアップ

このフォルダをClaude Codeで開いた後、以下を指示してください：

```
CLAUDE.mdを読んで、Next.js + TypeScript + Tailwindプロジェクトとして初期化して、
docs/prototype/のJSXプロトタイプを参考にSupabase連携の本番アプリを構築して
```

## フォルダ構成

```
tournament-system/
├── CLAUDE.md                ← Claude Codeへの指示書（自動で読まれる）
├── README.md                ← このファイル
├── docs/
│   ├── SPECIFICATION.md     ← 詳細仕様書
│   ├── HISTORY.md           ← 開発経過
│   ├── nipponkempo_proposal.pptx ← 提案資料
│   └── prototype/
│       └── nipponkempo-tournament-v2.2.jsx ← プロトタイプ
└── src/
    └── lib/
        └── types.ts         ← TypeScript型定義
```
