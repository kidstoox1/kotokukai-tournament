## 会話の経緯

### v1.0 — 初回プロトタイプ
- 4画面構成（管理・審判入力・モニター・観覧）
- シンプルなトーナメントのみ
- 学年×性別で均等にカテゴリ分け

### v2.0 — リーグ→トーナメント対応
- 孝徳会から実際の大会資料（申込書+対戦表PDF）を受領
- 実際のカテゴリ構成に合わせて修正（合同カテゴリ、面の種類等）
- リーグ戦→トーナメントの2段階進行を実装
- 試合記録の詳細化（本数・警告・引き分け・不戦勝・失格）
- 先方フィードバック：「審判ではなく記録係が操作する」→ UI名称変更

### v2.1 — フィードバック反映
- 白を白色で表現（青→白に変更）
- 柔軟なフェーズ進行（リーグ→リーグ / リーグ→トーナメント / 予選→決勝）
- 警告の自動加算を明確化（確定スコアプレビュー）

### v2.2 — 管理機能強化
- カテゴリ別の開始形式選択（リーグ戦/トーナメント/予選トーナメント）
- コート割り当て機能
- グループ情報の明示（記録係画面で「小学3年男子 Aグループ」表示）
- 進出人数の設定（デフォルト1名）
- 3位決定戦の有無設定（デフォルトON）
- 3位決定戦ロジック：2グループ×各1名進出→各グループ2位同士
- 最終順位表示（🥇🥈🥉）
- 試合結果の修正機能（管理画面・記録係画面の両方）
- 記録係画面での試合種別明示（🏆決勝 / 🥉3位決定戦 / Aグループ）
- 3位決定戦を決勝より先に並べるソート
- **リーグ決勝フェーズ追加**（旧リーグ2次を廃止、上位者で総当たり→順位確定）
- 同率順位対応（勝ち点+本数差が同じ→同順位）
- フェーズを戻す機能（確認ダイアログ付き）

## 先方からの重要フィードバック

1. **1コートに記録係含めスタッフ5名**。記録係がタブレット操作。
2. **リーグ戦が基本**。学年ごとに総当たり→人数によってリーグ数が変わる→トーナメント or リーグ決勝に進む。
3. **試合記録**: 2本先取3本勝負、警告2回→相手に1本、不戦勝→2-0、失格→0-2。
4. **リーグ戦で時間切れは「引き分け」**。
5. **保護者もスマホで見たい** → 同時接続対策が必要。
